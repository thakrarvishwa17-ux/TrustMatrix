# TrustMatrix
TrustMatrix is an AI-driven security platform that monitors user behavior, device changes, and risk signals in real time to prevent account takeover fraud while maintaining a seamless user experience.
AI-powered fraud detection and trust scoring system
Built using Flask,Python,HTML/CSS, and machine learning
Predicts fraud probability using device, SIM, location, OTP, and transaction risk signals
Developed by: Vishwa Thakrar and Ruchi Parmar
